Connecting

Arda Marred was designed using Mudlet and is one of the top MUD clients today. There are other ways to connect to Arda Mard than to use Mudlet or another MUD client. Players can connect to Arda Marred by using the following information:

Address: 159.223.146.67
Port: 4000

  1. Open Mudlet and click the connect button.

  2. Add a new profile to connect with.

  3. Add a new profile to connect with, change the name, server address and port number.

  4. Click Connect.
